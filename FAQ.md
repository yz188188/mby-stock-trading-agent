# FAQ

## 这个仓库有源码吗？

没有。

本仓库是公开观察手册和产品方法论文档，不包含喵板鸭后端源码、前端源码、接口、数据库结构、采集脚本或任何 token/API key。

## 这是荐股工具吗？

不是。

喵板鸭整理的是盘中异动和观察清单，不推荐具体股票，不提供买点、卖点或仓位建议。

## 这是自动交易机器人吗？

不是。

`mby-stock-trading-agent` 只是 GitHub repo 名称。仓库内容不包含自动交易程序，也不提供交易执行能力。

## 为什么关注 9:25 和 9:30–9:40？

因为这两个阶段信息密度高，资金攻击方向更容易提前暴露。

但早盘信号只代表“值得观察”，不代表“应该买入”。
