# 主线板块与题材集中

盘中个股异动要放到板块结构里看。

单只股票拉升不一定有意义；多个同方向股票集中异动，才更可能说明资金正在攻击某个题材。

## 喵板鸭关注

- 同题材是否多股异动；
- 龙头、前排、后排是否同步；
- 是否从老主线切换到新方向；
- 是否只是分支补涨；
- 是否出现主线扩散。

## 判断问题

1. 这是主线，还是分支？
2. 是新方向启动，还是旧方向反抽？
3. 是前排强，还是后排跟风？
4. 是否已经过度拥挤？

主线判断比单个信号更重要。
