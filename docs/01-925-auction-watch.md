# 9:25 竞价异动怎么看？

集合竞价不是正式开盘，但它经常提前暴露资金态度。

喵板鸭关注的不是“谁高开最多”，而是：

- 高开是否有成交支撑；
- 是否来自昨日强势方向；
- 是否存在弱转强；
- 是否有同题材个股一起表现；
- 是否只是单只股票孤立异动。

9:25 的作用不是给买点，而是帮助用户建立当天第一份观察清单。

## 观察顺序

1. 先看是否有明显题材集中；
2. 再看竞价强度是否和昨日市场结构有关；
3. 再看成交是否支持这个强度；
4. 最后判断它是主线回流、分支补涨，还是孤立异动。

## 常见误区

- 只看高开幅度；
- 看到竞价强就默认会继续涨；
- 不看市场情绪阶段；
- 不看板块配合；
- 把竞价异动当成交易指令。

竞价异动只能说明“值得观察”，不能说明“应该买入”。
