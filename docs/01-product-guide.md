# 喵板鸭使用指南：谁最适合用？

先说结论：喵板鸭不是给所有人准备的。

它更适合两类用户。

## 第一类：懂市场情绪的人

如果你能判断当前市场处在冰点、分歧、修复还是高潮，喵板鸭会更有用。

它负责在盘中把放量拉升、竞价异动、题材共振的股票先筛出来。

你负责判断：这些信号是不是出现在合适的情绪节点里。

工具不替你决策，但能帮你少翻很多无效信息。

## 第二类：有基本面和题材理解的人

如果你本来就懂方向、懂逻辑、懂产业链，MBY 股票开盘观察 Agent 更像一个盘中雷达。

它每天帮你从全市场里筛出最强势的一批票。

你再结合自己的基本面判断，挑出真正符合逻辑的方向。

## 产品本质

### 它擅长什么？

- 9:25 提示竞价异动
- 9:30-9:40 自动扫描盘中强势信号
- 帮你节省手动翻 5000 只股票的时间
- 辅助观察题材热度、资金方向和盘中强弱变化
- 盘后结合历史数据持续优化信号框架

### 它不做什么？

- 不做基本面分析
- 不判断个股能走多远
- 不提供买卖点
- 不承诺收益
- 不替你做交易决策

## 正确打开方式

MBY 股票开盘观察 Agent 负责回答：

> 当下市场里，谁正在变强？

你负责回答：

> 这个方向值不值得继续跟踪？

工具负责筛选，判断仍然属于你。

只看信号、不看位置、不看情绪、不看逻辑，风险会很高。
