## 喵板鸭关注什么？

### 1. 9:25 竞价异动

集合竞价结束后，市场会提前暴露一批异常信号：

- 高开强度；
- 竞价成交额；
- 昨日强势股承接；
- 弱转强迹象；
- 题材方向是否集中。

喵板鸭会在 9:25 后自动整理竞价异动，帮助用户知道：今天开盘前，哪些方向已经先动了。

### 2. 9:30–9:40 开盘强势信号

开盘后的前 10 分钟，是市场资金最容易暴露攻击方向的阶段。

喵板鸭会自动整理：

- 放量拉升；
- 快速上攻；
- 题材集中；
- 弱转强；
- 主线扩散；
- 开盘强度靠前的股票。

这些信号不是买卖建议，而是帮助用户快速建立当天观察列表。

### 3. 盘中监控

盘中继续观察：

- 信号是否扩散；
- 强势股是否持续；
- 题材是否形成合力；
- 是否出现冲高回落；
- 市场情绪是修复、分歧还是退潮。

### 4. 盘后结构化复盘

收盘后，喵板鸭会回看：

- 早盘信号是否延续；
- 哪些方向冲高回落；
- 哪些题材形成主线；
- 当天情绪是高潮、分歧还是退潮；
- 明天应该重点观察哪些方向。
